# AutoDoctor — статический лендинг

Полностью автономный лендинг без сборки, npm, внешних библиотек и трекеров.

## Локальный запуск

Можно открыть `index.html` напрямую в браузере.

Для проверки через локальный HTTP-сервер:

```powershell
cd lending
python -m http.server 8080
```

После этого откройте `http://localhost:8080`.

## Публикация

Загрузите содержимое папки `lending` в корень любого статического хостинга (GitHub Pages, Cloudflare Pages, Netlify, S3 или обычный веб-сервер). Команда сборки и серверная часть не нужны.

Канонический адрес страницы задан как `https://autodoctor.by/`. Если лендинг будет опубликован на поддомене или в другом пути, обновите `canonical` и `og:url` в `index.html`.

Форма не отправляет данные на сервер: после валидации она открывает подготовленное письмо на `pilot@autodoctor.by` в почтовом приложении пользователя. Перед публикацией убедитесь, что этот почтовый ящик создан.
